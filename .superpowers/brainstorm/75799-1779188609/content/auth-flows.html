B — With app lock (opt-in)

User enables "App lock" in Profile › Privacy. After 1 min in background, shows a lock screen (dark, minimal) with biometric prompt auto-triggering. Touch ID / Face ID, with password fallback. Same login as A — biometric is never part of login.

Both flows share the same login screen — only difference from signup is no name field. Biometrics only appear in Flow B, on every reopen once enabled.
