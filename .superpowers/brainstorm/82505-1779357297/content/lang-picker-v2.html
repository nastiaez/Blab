A — List + search + checkmark (Recommended)

Apple Settings / WhatsApp pattern. Flag + name in a clean list. Search at top. Popular languages pinned first, rest alphabetical below. Checkmark + light purple row highlight on selection. Button grey (inactive) until pick — then purple.

Same design on both "New chat" and Aswin's screen. Same component, same order, same search.

B — Duolingo grid cards

2-column grid, big flag + name. Selected card gets purple border + tint. Visually engaging. No search — but hard to scale past 12 languages. Works great for small curated lists.

Problem: "New chat" screen has 18+ languages — grid gets very long. No search.

C — Minimal, text only, no flags

Google Translate approach — text list, search prominent. No flags (avoids flag-vs-language ambiguity). Clean, fast to scan. Slightly less warm/visual than A.

All options: no pre-selection, button grey until pick, same component used in both "New chat" (Phone 2) and Aswin's onboarding (Phone 4). Language order: top 8 popular pinned, rest alphabetical below a divider.
