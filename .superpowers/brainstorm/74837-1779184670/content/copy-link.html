✅ Link copied
Sheet closes. Toast fades after 2s. User stays on invite screen to go back manually.

A — Toast + stay on screen

Dark toast "Link copied" appears, sheet closes, user stays on the invite screen. They go back whenever ready. Standard pattern (iOS, Android).

Row changes to green "Copied! · Paste it in a chat with your friend". Sheet stays open briefly, user taps Done to close. Gives clear next-step instruction.
