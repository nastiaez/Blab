Playful — which flavor?

Playful is broad. Pick the modifier that sharpens it. This sets the bar for color saturation, type quirk, and copy tone.

😏

Witty

Clever + dry. Smart jokes in microcopy, not stickers. Confident playful. Think Mailchimp, Notion, Linear.

🧸

Cozy

Cute + soft. Rounded everything, pastel tones, mascots welcome. Think Cake, Duolingo (soft side), Bumble.

⚡

Bold

Loud + energetic. Big type, saturated color, expressive. Think BeReal, Poolsuite, Arc browser.

🤓

Nerdy

Curious + craft. Reference-rich, language nerd in-jokes, dictionary energy. Think Are.na, Readwise, Anki done well.
